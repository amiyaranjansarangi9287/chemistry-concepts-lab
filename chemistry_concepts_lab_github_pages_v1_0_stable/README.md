# Chemistry Concepts Lab — Class 1–10

A static, GitHub Pages-ready Chemistry learning lab for Class 1–10.

## What is included

- `index.html` — primary GitHub Pages entry file
- `chemistry_concepts_lab_stable.html` — stable named copy of the same lab
- `versions/chemistry_concepts_lab_v1_8.html` — preserved versioned build
- `docs/DEPLOYMENT.md` — deployment instructions
- `docs/RELEASE_NOTES.md` — release summary

## Recommended GitHub Pages setup

Upload the contents of this folder to a GitHub repository and enable GitHub Pages from the repository root.

The page will open from:

```text
https://<your-github-username>.github.io/<repository-name>/
```

No backend, package manager, build command, database, or API key is required.

Chemistry Concepts Lab v1.8 packaged as v1.0 Stable for deployment.
